# 🌸 Eunoia
A minimalist social media web app — For people to interact with their friends, family or even strangers and share their life stories!


### Features
- 📝 Sign up and log in
- 👋 Personalized homepage greeting
- 🚪 Logout system
- ⚡ LocalStorage-based session handling
